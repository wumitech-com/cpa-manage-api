  • 单设备多次注册流程：循环执行（启动云手机 → ResetPhoneEnv（重置+换机） → 安装TikTok APK → 执行注册脚本）
  • 同一台设备将重复执行完整流程，每次注册一个新的账号

TikTok APK安装目录，位于 /apk_repo/ 下
ResetPhoneEnv参数配置
为空时从phoneId自动提取
Android SDK版本
为空时自动获取
为空时从phoneId自动提取
为空时将随机选择：ipidea、closeli、ipbiubiu

可选参数
说明：
  • 多设备并行注册流程：多个设备同时执行（启动云手机 → ResetPhoneEnv（重置+换机） → 安装TikTok APK → 执行注册脚本）
  • 每个设备可以注册多个账号（通过"每个设备目标账号数"设置），多个设备并行执行，提高注册效率
