# 部署命令步骤

## 1. 连接到跳板机
```powershell
ssh -i $env:USERPROFILE\.ssh\id_rsa ubuntu@106.75.152.136
```

## 2. 连接到目标服务器
```bash
ssh ubuntu@10.13.135.74
```

## 3. 停止旧进程
```bash
ps aux | grep java | grep cpa-manage-api | grep -v grep | awk '{print $2}' | xargs kill -9 2>/dev/null || sudo pkill -f cpa-manage-api
```

或者更简单的方式：
```bash
sudo pkill -f cpa-manage-api
```

## 4. 复制jar文件并启动应用
```bash
sudo mkdir -p /opt/cpa-manage-api
sudo cp /tmp/cpa-manage-api-0.0.1-SNAPSHOT.jar /opt/cpa-manage-api/
cd /opt/cpa-manage-api
sudo nohup java -jar cpa-manage-api-0.0.1-SNAPSHOT.jar --spring.profiles.active=dev > /var/log/cpa-manage-api.log 2>&1 &
```

## 5. 检查进程和日志
```bash
# 检查进程是否运行
ps aux | grep cpa-manage-api | grep -v grep

# 查看日志最后50行
tail -50 /var/log/cpa-manage-api.log

# 实时查看日志（可选）
tail -f /var/log/cpa-manage-api.log
```

## 一键执行所有命令（在目标服务器上）
```bash
sudo pkill -f cpa-manage-api; \
sleep 2; \
sudo mkdir -p /opt/cpa-manage-api; \
sudo cp /tmp/cpa-manage-api-0.0.1-SNAPSHOT.jar /opt/cpa-manage-api/; \
cd /opt/cpa-manage-api && \
sudo nohup java -jar cpa-manage-api-0.0.1-SNAPSHOT.jar --spring.profiles.active=dev > /var/log/cpa-manage-api.log 2>&1 & \
sleep 3 && \
ps aux | grep cpa-manage-api | grep -v grep && \
tail -20 /var/log/cpa-manage-api.log
```
