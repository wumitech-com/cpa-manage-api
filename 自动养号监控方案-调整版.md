# 自动养号任务监控方案（无日志版本）

## 🔍 问题分析

**核心问题：** 自动养号任务（TTTest_main_auto_1.py）不输出日志或日志输出很少

**原Shell脚本局限性：**
- ❌ 依赖日志文件修改时间判断
- ❌ 无法准确判断任务是否正常执行
- ❌ 可能误判（任务运行但不输出日志）

**需要新的监控指标：**
- ✅ 进程状态（进程是否存在）
- ✅ Screen会话状态
- ✅ 数据库操作时间（养号任务的实际产出）
- ✅ 心跳机制（任务主动上报）

---

## 🎯 推荐方案对比

### 方案A：基于进程和数据库监控 ⭐⭐⭐ 推荐

**监控指标：**
1. 进程是否存在
2. Screen会话是否存在
3. 数据库最后更新时间
4. 可选：HTTP心跳接口

**判断逻辑：**
```
进程存在 + 数据库有更新 = 正常运行 ✅
进程存在 + 数据库长时间无更新 = 任务卡死 ⚠️
进程不存在 = 已崩溃 ❌ 自动重启
```

### 方案C：基于数据库变化监控 ⭐⭐⭐

**核心思想：** 养号任务最终会操作数据库，监控数据库变化即可

**监控SQL：**
```sql
-- 查询设备最后一次操作时间
SELECT 
    device_id,
    MAX(update_time) as last_update,
    COUNT(*) as operation_count,
    TIMESTAMPDIFF(SECOND, MAX(update_time), NOW()) as seconds_ago
FROM tt_account_data
WHERE device_id = '10.7.136.129'
GROUP BY device_id;

-- 判断逻辑
-- seconds_ago < 600  → 正常运行（10分钟内有操作）
-- seconds_ago > 1800 → 可能异常（30分钟没操作）
-- seconds_ago > 3600 → 已停止（1小时没操作）
```

**Java实现：**
```java
@Scheduled(fixedRate = 300000)
public void checkDatabaseActivity() {
    for (String deviceId : monitorDevices) {

        if (!hasProcess) {
            // 进程不存在，直接重启
            restartScript(deviceId);
        } else if (secondsAgo > 3600) {
            // 进程存在但1小时没操作，可能卡死，重启
            restartScript(deviceId);
        }
    }
}
```

## 📊 最终推荐方案（综合方案）

### 方案D：多维度监控 ⭐⭐⭐⭐⭐ 最佳

**综合以下指标：**

| 监控维度 | 检查方式 | 权重 | 说明 |
|---------|---------|------|------|
| **进程状态** | SSH: ps aux \| grep | 高 | 进程不存在→直接重启 |
| **Screen会话** | SSH: screen -list | 中 | 辅助确认 |
| **数据库活跃度** | SQL: 查最后更新时间 | 高 | 核心指标 |
| **心跳上报** | Redis: 心跳时间戳 | 高 | 最准确（需改造脚本） |
| **资源占用** | SSH: top/CPU/内存 | 低 | 判断是否卡死 |

### 判断逻辑流程图

```
开始检查
   ↓
1. 检查进程是否存在？
   ├─ 否 → ❌ 已崩溃 → 立即重启
   └─ 是 → 继续
   ↓
2. 检查Screen会话是否存在？
   ├─ 否 → ⚠️ 异常（可能被手动关闭） → 记录警告
   └─ 是 → 继续
   ↓
3. 查询数据库最后操作时间
   ├─ < 10分钟  → ✅ 正常运行
   ├─ 10-30分钟 → ⚠️ 可能变慢，继续观察
   ├─ 30-60分钟 → ⚠️ 异常，发送警告
   └─ > 60分钟  → ❌ 任务卡死 → 重启
   ↓
4. （可选）检查CPU/内存
   ├─ CPU > 95% 持续10分钟 → 可能死循环 → 重启
   └─ 内存 > 90% → 可能内存泄漏 → 重启
   ↓
记录监控结果到数据库
```

### 2. Java监控服务（核心代码）

```java
@Service
@Slf4j
public class NurtureTaskMonitorService {
    
    @Autowired
    private SshUtil sshUtil;
    
    @Autowired
    private TtAccountDataRepository accountDataRepository;
    
    @Autowired
    private ScriptMonitorLogRepository monitorLogRepository;
    
    /**
     * 每5分钟检查一次
     */
    @Scheduled(fixedRate = 300000)
    public void monitorNurtureTasks() {
        log.info("=== 开始检查养号任务状态 ===");
        
        for (MonitorConfig config : getMonitorConfigs()) {
            try {
                checkNurtureTask(config);
            } catch (Exception e) {
                log.error("检查任务失败: {}", config.getDeviceId(), e);
            }
        }
    }
    
    /**
     * 检查单个养号任务
     */
    public ScriptMonitorLog checkNurtureTask(MonitorConfig config) {
        log.info(">>> 检查设备: {}", config.getDeviceId());
        
        ScriptMonitorLog log = new ScriptMonitorLog();
        log.setDeviceId(config.getDeviceId());
        log.setScriptName(config.getScriptName());
        log.setCheckTime(LocalDateTime.now());
        
        // 1. 检查进程状态
        boolean hasProcess = checkProcess(config);
        log.setHasProcess(hasProcess);
        
        if (!hasProcess) {
            log.setStatus("STOPPED");
            log.setRemark("进程不存在");
            log.setRestarted(true);
            restartScript(config);
            monitorLogRepository.save(log);
            return log;
        }
        
        // 2. 检查Screen会话
        boolean hasScreen = checkScreen(config);
        log.setHasScreen(hasScreen);
        
        // 3. 检查数据库活跃度（关键！）
        Integer dbIntervalSeconds = checkDatabaseActivity(config.getDeviceId());
        log.setDbOperationInterval(dbIntervalSeconds);
        
        if (dbIntervalSeconds != null) {
            LocalDateTime lastOpTime = LocalDateTime.now()
                .minusSeconds(dbIntervalSeconds);
            log.setLastDbOperationTime(lastOpTime);
        }
        
        // 4. 判断状态
        String status = determineStatus(hasProcess, hasScreen, dbIntervalSeconds);
        log.setStatus(status);
        
        // 5. 决定是否重启
        boolean needRestart = shouldRestart(status, dbIntervalSeconds);
        
        if (needRestart) {
            log.info("⚠️ 任务异常，准备重启: {}", config.getDeviceId());
            boolean success = restartScript(config);
            log.setRestarted(success);
            log.setRemark(success ? "已自动重启" : "重启失败");
        }
        
        // 6. 保存监控记录
        monitorLogRepository.save(log);
        return log;
    }
    
    /**
     * 检查数据库活跃度（核心方法）
     */
    private Integer checkDatabaseActivity(String deviceId) {
        try {
            // 查询该设备最后一次操作时间
            // 假设 tt_account_data 表有 device_id 和 update_time 字段
            Object[] result = accountDataRepository
                .findLastOperationTime(deviceId);
            
            if (result != null && result.length > 0) {
                LocalDateTime lastUpdateTime = (LocalDateTime) result[0];
                Duration duration = Duration.between(lastUpdateTime, LocalDateTime.now());
                return (int) duration.getSeconds();
            }
            
            return null;
        } catch (Exception e) {
            log.error("查询数据库活跃度失败", e);
            return null;
        }
    }
    
    /**
     * 判断任务状态
     */
    private String determineStatus(Boolean hasProcess, Boolean hasScreen, 
                                   Integer dbIntervalSeconds) {
        // 进程不存在
        if (!hasProcess) {
            return "STOPPED";
        }
        
        // 数据库无法查询
        if (dbIntervalSeconds == null) {
            return "UNKNOWN";
        }
        
        // 根据数据库活跃度判断
        if (dbIntervalSeconds < 600) {  // 10分钟内
            return "RUNNING";
        } else if (dbIntervalSeconds < 1800) {  // 30分钟内
            return "SLOW";
        } else if (dbIntervalSeconds < 3600) {  // 1小时内
            return "STUCK";
        } else {
            return "STOPPED";
        }
    }
    
    /**
     * 判断是否需要重启
     */
    private boolean shouldRestart(String status, Integer dbIntervalSeconds) {
        // 状态为STOPPED，直接重启
        if ("STOPPED".equals(status)) {
            return true;
        }
        
        // 数据库1小时没更新，认为卡死，重启
        if (dbIntervalSeconds != null && dbIntervalSeconds > 3600) {
            return true;
        }
        
        return false;
    }
    
    /**
     * 检查进程是否存在
     */
    private boolean checkProcess(MonitorConfig config) {
        try {
            String command = String.format(
                "ps aux | grep %s | grep -v grep | wc -l", 
                config.getScriptName()
            );
            
            SshUtil.SshResult result = sshUtil.executeCommand(
                config.getDeviceId(), command
            );
            
            return result.isSuccess() && 
                   Integer.parseInt(result.getOutput().trim()) > 0;
        } catch (Exception e) {
            log.error("检查进程失败", e);
            return false;
        }
    }
    
    /**
     * 检查Screen会话
     */
    private boolean checkScreen(MonitorConfig config) {
        try {
            String command = String.format(
                "screen -list | grep %s | wc -l", 
                config.getScreenName()
            );
            
            SshUtil.SshResult result = sshUtil.executeCommand(
                config.getDeviceId(), command
            );
            
            return result.isSuccess() && 
                   Integer.parseInt(result.getOutput().trim()) > 0;
        } catch (Exception e) {
            log.error("检查Screen失败", e);
            return false;
        }
    }
    
    /**
     * 重启脚本
     */
    private boolean restartScript(MonitorConfig config) {
        log.warn(">>> 重启脚本: {}", config.getDeviceId());
        
        try {
            String command = String.format(
                "screen -S %s -X quit 2>/dev/null || true; " +
                "screen -S %s -X quit 2>/dev/null || true; " +
                "sleep 1; " +
                "screen -S %s -dm; " +
                "screen -S %s -X stuff 'cd %s && python3 %s.py\\n'",
                config.getScreenName(), config.getScreenName(),
                config.getScreenName(), config.getScreenName(),
                config.getScriptPath(), config.getScriptName()
            );
            
            SshUtil.SshResult result = sshUtil.executeCommand(
                config.getDeviceId(), command
            );
            
            if (result.isSuccess()) {
                log.info("✅ 重启成功: {}", config.getDeviceId());
                return true;
            } else {
                log.error("❌ 重启失败: {}", result.getError());
                return false;
            }
        } catch (Exception e) {
            log.error("重启异常", e);
            return false;
        }
    }
}
```



### 策略3：进程 + 数据库 + 心跳 ⭐ 最佳

```
优点：最准确、实时性最好

```
